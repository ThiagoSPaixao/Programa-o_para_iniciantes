Tomada de Decisão — IF / ELSE


Olá, programador em crescimento! 🌱

Hoje vamos aprender sobre tomada de decisão — uma habilidade que você usa o tempo todo sem perceber! Desde "vou levar guarda-chuva?" até "qual rota pegar para o trabalho?", sua vida é feita de decisões.

Na programação, chamamos isso de IF/ELSE (SE/SENÃO), e é incrivelmente simples e poderoso! Você vai descobrir que já pensa como um programador há anos — só não sabia o nome técnico! 😄

  • ✅ Criar seus próprios exemplos com base no dia a dia

🧠 1. O que é uma condição?

Uma condição é uma pergunta que só pode ter duas respostas: Verdadeiro ou Falso.

Na vida real:

  • Se o semáforo estiver verde, posso atravessar.
  • Se o saldo for maior que o valor da compra, posso pagar
  • Se tiver café, faço café; senão, vou de água mesmo ☕💧

💻 2. O “if” em programação

Na programação, usamos if para expressar essa lógica.

🧩 Estrutura básica:


python

if condição:
    ação_se_verdadeiro
else:
    ação_se_falso

🧠 Exemplo prático:



python

hora = 13

if hora < 12:
    print("Bom dia!")
else:
    print("Boa tarde!")

💬 O que acontece aqui?

O programa verifica se a hora é menor que 12.
Se for, diz “Bom dia!”.
Se não for, diz “Boa tarde!”.


🚦 3. Condições com comparações

Podemos comparar valores usando operadores:

Símbolo Significado Exemplo
== Igual a if idade == 18:
!= Diferente de if cor != "vermelho":
> Maior que if nota > 7:
< Menor que if temperatura < 20:
>= Maior ou igual a if saldo >= 0:
<= Menor ou igual a if tentativas <= 3:

Esses operadores são o alfabeto das decisões.


⚙️ 4. Exemplo com a vida real

Imagine que você quer decidir se pode sair para passear:


python

chovendo = True

if chovendo:
    print("Leve o guarda-chuva!")
else:
    print("Aproveite o sol!")

➡️ O programa pensa como você pensaria.

Se estiver chovendo (condição verdadeira), ele te dá um aviso.
Se não, muda o plano.


🧩 5. Exemplo prático com entrada de dados

Vamos deixar o usuário tomar decisões com o programa:


python

idade = int(input("Digite sua idade: "))

if idade >= 18:
    print("Você pode dirigir!")
else:
    print("Você ainda não pode dirigir.")

Esse é o tipo de código que você verá em cadastros, sistemas e formulários — ele decide o que fazer com base em informações do usuário.
