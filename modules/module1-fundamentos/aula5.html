🧠 1. O que é um Laço de Repetição?

Um laço de repetição é uma estrutura que faz o computador executar o mesmo bloco de código várias vezes.

💬 Na vida real:

  • Você escova os dentes enquanto o cronômetro não chega a 2 minutos.
  • Você ouve músicas até o sono bater.
  • Você tenta ligar o celular enquanto ele não liga.

Essas são situações onde uma ação é repetida até que algo mude — e é isso que faremos no código.


⚙️ 2. O Laço FOR — “Repita X Vezes”

Use o for quando souber quantas vezes quer repetir algo.

🧩 Estrutura:



python

for número in range(5):
print("Repetindo...")

💬 Explicando:

  • range(5) gera uma sequência de 0 a 4 → total de 5 repetições.
  • O for repete o bloco uma vez para cada número.

🧠 Exemplo Prático:


python

for i in range(1, 6):
print("Contagem:", i)

➡️ Saída:


                
Contagem: 1
Contagem: 2
Contagem: 3
Contagem: 4
Contagem: 5
